**The RealTime FullStack Chat App** is a seamless integration of *Frontend* and *Backend* technologies, delivering instant data updates via **Socket.io**

**TECHNOLOGY STACK**

**Frontend:** React, React Router, Cloudinary
**Backend:** Node.js, Express, MongoDB, Socket.io
**Deployment:** Render

The App is accessible at the following website: 
__https://realtime-fullstack-chat-app-ub45.onrender.com__
