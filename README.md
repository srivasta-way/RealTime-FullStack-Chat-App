RealTime FullStack Chat App is the final product of seamless integration in the frontend and backend technologies that offers a realTime updation of data using socket.io package in the backend of the project.

Technology used:
Frontend: React, React-Router, cloudinary, 
Backend: Express, NodeJS, MongoDB, Render(for deployment) ,Socket.io

The App is accessible at the following website: 
https://realtime-fullstack-chat-app-ub45.onrender.com
